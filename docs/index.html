Welcome to codedevils.org documentation!¶

The CodeDevils website is the face of the organization and serves as a portal into all things CodeDevils. The website is built using Python Django 3.0, and supports user authentication, member registration and communication with external users. The website is open source and can be found on our GitHub page.
